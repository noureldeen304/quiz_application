Welcome to the Application, here are the endpoints available for you

Endpoint Description
GET /api/v1/admin/quizzes Retrieves a list of all quizzes.
GET /api/v1/admin/quizzes/my-quizzes Retrieves a list of quizzes created by the admin.
GET /api/v1/admin/quizzes/id/{id} Retrieves a specific quiz by its ID.
GET /api/v1/admin/quizzes/category/{category} Retrieves quizzes based on a specified category.
GET /api/v1/admin/quizzes/category/{category}/version/{version} Retrieves quizzes based on a specified category and version.
GET /api/v1/admin/quizzes/id/{id}/model-answers Retrieves model answers for a specific quiz.
GET /api/v1/admin/quizzes/id/{id}/students-scores Retrieves students' scores for a specific quiz.
POST /api/v1/admin/quizzes/create-quiz-with-random-questions Creates a new quiz with a specified category, number of questions, and version.
POST /api/v1/admin/quizzes/create-quiz-with-selected-questions Creates a new quiz with a specified category, version, and selected question IDs.
PUT /api/v1/admin/quizzes/update-quiz/{id}/update-version Updates the version of a specific quiz.
PUT /api/v1/admin/quizzes/update-quiz/{id}/add-random-questions Adds a specified number of random questions to a specific quiz.
PUT /api/v1/admin/quizzes/update-quiz/{id}/add-selected-questions Adds selected questions to a specific quiz based on question IDs.
PUT /api/v1/admin/quizzes/update-quiz/{id}/remove-random-questions Removes a specified number of random questions from a specific quiz.
PUT /api/v1/admin/quizzes/update-quiz/{id}/remove-selected-questions Removes selected questions from a specific quiz based on question IDs.
PUT /api/v1/admin/quizzes/update-quiz/{id}/remove-all-questions Removes all questions from a specific quiz.
DELETE /api/v1/admin/quizzes/remove-quiz/{id} Removes a specific quiz by its ID.
