Welcome to the Application, here are the endpoints available for you

Endpoint Description
/api/v1/admin/questions Retrieve the list of all questions
/api/v1/admin/questions/id/{id} Retrieve the list of questions by ID
/api/v1/admin/questions/category/{category} Retrieve the list of questions by category name
/api/v1/admin/questions/difficultylevel/{difficultylevel} Retrieve the list of questions by difficulty level
/api/v1/admin/questions/addquestion Add new question
/api/v1/admin/questions/modify/{id} Modify an existing question by it's ID
/api/v1/admin/questions/remove/{id} Remove a question by it's ID
/api/v1/admin/quizzes Retrieve the list of all quizzes created by you
/api/v1/admin/quizzes/category/{category} Retrieve the list of all questions with admin privileges
/api/v1/admin/quizzes/category/{category}/version/{version} Retrieve the list of all questions with admin privileges
/api/v1/admin/quizzes/create-quiz-with-random-questions Retrieve the list of all questions with admin privileges
/api/v1/admin/quizzes/create-quiz Retrieve the list of all questions with admin privileges
/api/v1/admin/quizzes/update-quiz/{id}/update-version Retrieve the list of all questions with admin privileges
/api/v1/admin/quizzes/update-quiz/{id}/add-random-questions Retrieve the list of all questions with admin privileges
/api/v1/admin/quizzes/update-quiz/{id}/remove-random-questions Retrieve the list of all questions with admin privileges
/api/v1/admin/quizzes/update-quiz/{id}/add-specific-questions Retrieve the list of all questions with admin privileges
/api/v1/admin/quizzes/update-quiz/{id}/remove-specific-questions Retrieve the list of all questions with admin privileges
/api/v1/admin/quizzes/update-quiz/{id}/remove-all-questions Retrieve the list of all questions with admin privileges
/api/v1/admin/quizzes/remove-quiz/{id} Retrieve the list of all questions with admin privileges
