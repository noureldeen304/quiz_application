Welcome to the Application, here are the endpoints available for you

Endpoint Description
/api/v1/student/quizzes/category/{categoryName} Retrieves quizzes based on the specified category name.
/api/v1/student/quizzes/my-quiz-scores Retrieves quiz scores and details for the currently authenticated student.
/api/v1/student/quizzes/id/{id}/take-quiz Retrieves details for a specific quiz based on the provided quiz ID.
/api/v1/student/quizzes/id/{id}/submit-answers Submits answers for a specific quiz based on the provided quiz ID. Returns the score after submission.
